# Sales-Job-Application-Automation
I do high-ticket sales and apply to 100 jobs a day for different businesses I work with, so I automated the job application for this instead of applying manually. I update the job codes every time I run it from the last job code I applied for to the most recent job code that has been listed 
START_CODE = "last job code I applied for" + 1
END_CODE = "most recent job code that has been listed"

and the code increments through these job codes
